# Experimental Design — Team Strategy Guide

---

## Recommended Role Split

| Role | Sections to Complete | Description of Responsibilities |
|:----:|:--------------------:|:--------------------------------|
| **Designer (40 pts)** | Problem Statement, Hypothesis, Variables, Materials, Procedure & Diagrams | Write the problem statement; write the if/then/because hypothesis; identify IV, DV, and controls; list materials; write step-by-step procedure with diagrams |
| **Data Lead (34 pts)** | Quantitative Data Table, Graph, Statistics, Experimental Error | Build the raw data table with units; create the graph (title, labels, scale, line of best fit); calculate all statistics (mean, median, mode, range, IQR, st. dev); identify experimental errors |
| **Analyst (44 pts)** | Qualitative Observations, CER — Data Trend/Outliers/Variation, Conclusion, Applications & Recommendations | Record qualitative observations during trials; write claim/evidence/reasoning; write conclusion; describe applications and future experiments |
