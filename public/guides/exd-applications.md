# Experiment Improvments, Applications & Future Use examples

---

| General Topic | Ways to Improve the Experiment | Practical Applications | Ideas for Future Experiments |
|:-------------:|:------------------------------:|:----------------------:|:----------------------------:|
| **Friction** | Test more surfaces (at least 5); do 5 trials per surface instead of 3; use a ruler to measure distance more carefully | Shoe grip on gym floors; car tires on wet roads | Does adding weight on top of the block change how far it slides? Does wetting the surface change friction? |
| **Ramp Motion** | Use more ramp heights (5+); measure time with a stopwatch to the nearest tenth of a second; make sure the ball starts from the exact same spot each time | Skateboard ramps; playground slides | Does the size of the ball change how fast it rolls? Does a heavier ball go farther at the bottom? |
| **Air Resistance** | Drop from a higher point for easier timing; test more paper shapes; have the same person time every drop | Parachutes; paper airplane design | Does poking holes in a paper parachute change the fall time? Does a bigger parachute always fall slower? |
| **Bounce Height** | Use a meter stick taped to the wall to read bounce height more accurately; drop the ball from exactly the same height each time; test on more surfaces | Basketball court flooring; athletic shoes | Does the temperature of the ball change how high it bounces? Does pumping more air into a ball make it bounce higher? |
| **Dissolving** | Use a thermometer to keep water temperature exact; stir the same number of times for each trial; test more temperatures | Making hot chocolate; dissolving medicine | Does crushing the solid into powder make it dissolve faster? Does stirring speed matter? |
| **Chemical Reactions** | Measure the vinegar and baking soda with measuring spoons for exact amounts; do more trials; test more amounts of baking soda | Baking a cake (baking soda makes it rise); volcano science models | Does warm vinegar react faster than cold? Does using more vinegar change how much fizz is produced? |
| **Heat / Cooling** | Read the thermometer every 30 seconds for a better graph; wrap cups in towels to block drafts; test more cup materials | Travel mugs keeping drinks hot; coolers keeping drinks cold | Does adding a lid slow down cooling? Does the color of the cup matter when sitting in sunlight? |
| **Magnetism** | Measure distance with a ruler at small intervals (every 0.5 cm); keep other metal objects away from the table; test more distances | Refrigerator magnets; magnetic phone mounts | Does stacking two magnets together make them stronger? Are all magnet shapes equally strong? |
| **Buoyancy** | Use a clear container with measurement lines to see float height; keep the water temperature the same; test more shapes with the same amount of clay | Life jackets; boat and ship hull shapes | Does adding salt to the water change how high an object floats? Does water temperature affect buoyancy? |
| **Sound / Pitch** | Use a free tuner app on a phone to measure pitch instead of guessing by ear; pull rubber bands to the same tightness each time; test more lengths | Guitar and ukulele string tuning; drum design | Does a thicker rubber band make a lower pitch? Does the size of the box under the rubber band change the sound? |
